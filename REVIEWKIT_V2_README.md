# 🌟 ReviewKit V2 - 링크인바이오 + 리뷰 플랫폼

**"웹사이트 없어도 OK! 인스타그램 프로필 하나면 충분합니다"**

## 🎯 V2의 핵심 변화

### Before (V1): 위젯 중심
```
타겟: 웹사이트가 있는 비즈니스 (10%)
문제: 대부분의 소상공인은 웹사이트가 없음
```

### After (V2): 링크인바이오 + 위젯 하이브리드
```
타겟: 모든 비즈니스 (100%)
해결: 독립 페이지 제공 + 위젯 옵션
```

## 🚀 새로운 기능

### 1. 독립 공개 페이지 (Link-in-Bio)

**자동 생성되는 미니 홈페이지**
```
https://reviewkit.com/홍대-지민-카페
```

**포함 내용:**
- ✅ 비즈니스 정보 (이름, 설명, 주소)
- ✅ 평균 별점 + 리뷰 개수
- ✅ 승인된 리뷰 목록
- ✅ [전화하기] [지도보기] [리뷰 남기기] 버튼
- ✅ 소셜 미디어 링크 (인스타, 페이스북, 유튜브)
- ✅ 영업시간, 연락처 정보
- ✅ 모바일 최적화

### 2. 비즈니스 설정 페이지

**대시보드 → 페이지 설정**

추가 가능한 정보:
- 📞 전화번호
- 📍 주소
- 🕐 영업시간
- 📷 인스타그램
- 👍 페이스북
- ▶️ 유튜브
- 🗺️ 지도 링크

### 3. 공개 링크 관리

**설정 페이지에서:**
- 🔗 공개 링크 복사
- 👁️ 미리보기
- 📋 한 번에 복사

## 📱 사용 시나리오

### Scenario 1: 동네 카페 (웹사이트 없음)

**지민씨 - 홍대 카페 주인**

```
Step 1: ReviewKit 가입
  ↓
Step 2: 비즈니스 생성 "홍대 지민 카페"
  → 자동 생성: reviewkit.com/홍대-지민-카페
  ↓
Step 3: 설정에서 정보 입력
  ✅ 전화: 010-1234-5678
  ✅ 주소: 서울 마포구 홍대입구역 3번 출구
  ✅ 영업시간: 월-금 09:00-22:00
  ✅ 인스타: @jimin_cafe
  ↓
Step 4: 링크 활용
  ✅ 인스타그램 프로필에 등록 (링크인바이오)
  ✅ 네이버 플레이스 "홈페이지"에 등록
  ✅ QR 코드로 테이블에 부착
  ✅ 카카오톡 프로필 상태메시지
  ↓
Step 5: 고객 경험
  손님이 링크 클릭
  → 카페 정보 + 리뷰 한눈에 확인
  → [전화하기] 클릭으로 바로 예약
  → [리뷰 남기기]로 30초 만에 리뷰 작성
```

### Scenario 2: 프리랜서 (포트폴리오 없음)

**수진씨 - 프리랜서 디자이너**

```
Step 1: "수진 디자인 스튜디오" 생성
  → reviewkit.com/수진-디자인-스튜디오
  ↓
Step 2: 정보 입력
  ✅ 설명: "브랜드 아이덴티티 전문 디자이너"
  ✅ 포트폴리오: behance.net/sujin
  ✅ 인스타: @sujin_design
  ↓
Step 3: 링크 활용
  ✅ 링크드인 프로필
  ✅ 이메일 서명
  ✅ 명함 QR 코드
  ↓
Step 4: 클라이언트 경험
  링크 클릭
  → 수진님 소개 + 고객 추천사 확인
  → 신뢰도 ↑
  → 프로젝트 문의 ↑
```

### Scenario 3: 기존 웹사이트 있는 비즈니스

**기존 V1 기능 그대로 사용**

```
독립 페이지: reviewkit.com/business-name
  → 인스타그램, SNS에 공유
  
위젯 임베드: <script> 태그
  → 기존 웹사이트에 리뷰 섹션 추가
  
둘 다 사용 가능! ✅
```

## 🚀 사용 방법

### 1. 비즈니스 생성
```
http://localhost:8080/reviewkit/dashboard
→ "새 비즈니스 만들기"
→ 이름, 설명, 전화번호, 주소 입력
```

### 2. 페이지 설정
```
비즈니스 상세 → "⚙️ 페이지 설정"
→ 영업시간, 소셜 링크 추가
→ 저장
```

### 3. 공개 링크 활용
```
설정 페이지에서 링크 복사
→ http://localhost:8080/reviewkit/홍대-지민-카페

활용처:
✅ 인스타그램 프로필 (링크인바이오)
✅ 네이버 플레이스 홈페이지
✅ 카카오톡 프로필
✅ QR 코드 (테이블, 명함)
✅ 이메일 서명
```

### 4. 리뷰 수집
```
고객이 공개 페이지 방문
→ [⭐ 리뷰 남기기] 클릭
→ 30초 만에 리뷰 작성
→ 대시보드에서 승인
→ 자동으로 공개 페이지에 표시
```
