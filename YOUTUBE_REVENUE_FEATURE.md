# 💰 YouTube 수익 예측 기능

## 개요

YouTube 인기 영상 페이지에 **예상 광고 수익**과 **수익 잠재력 점수**를 추가했습니다.
크리에이터들이 "이 스타일로 영상을 만들면 얼마나 벌 수 있는지" 한눈에 파악할 수 있어요!

## 🎯 핵심 기능

### 1. 예상 광고 수익 💵
- **실시간 계산**: 조회수 × 카테고리별 CPM
- **카테고리별 CPM**: 과학/기술(8천~2.5만원) > 교육(6천~2만원) > 브이로그(4천~1.2만원)
- **범위 표시**: 최소/최대/평균 수익 모두 표시

### 2. 수익 잠재력 점수 📊
- **0~100점 척도**로 수익성 평가
- **계산 공식**: 참여율(40%) + 바이럴지수(30%) + CPM등급(30%)
- **5단계 등급**: 매우높음(80+) / 높음(60+) / 보통(40+) / 낮음(20+) / 매우낮음

### 3. 월 수익 시뮬레이션 📈
- **3가지 시나리오**: 주 1회 / 주 2~3회 / 거의 매일
- **예상 월 수익**: 영상당 수익 × 월 영상 개수
- **성장 가능성**: 바이럴 잠재력 기반 평가

## 📈 수익 잠재력 점수 계산 로직

### 1. 참여율 점수 (40%)
```
참여율 = (좋아요 + 댓글) / 조회수 × 100
점수 = min(100, (참여율 / 10) × 100)

예시:
- 참여율 10% → 100점
- 참여율 5% → 50점
- 참여율 2% → 20점
```

### 2. 바이럴 점수 (30%)
```
바이럴 지수 = 조회수 / 구독자수 × 100
점수 = min(100, (바이럴 지수 / 500) × 100)

예시:
- 바이럴 500% → 100점
- 바이럴 250% → 50점
- 바이럴 100% → 20점
```

### 3. CPM 점수 (30%)
```
점수 = min(100, (CPM / 25000) × 100)

예시:
- CPM 25,000원 → 100점
- CPM 12,500원 → 50점
- CPM 5,000원 → 20점
```

### 최종 점수
```
총점 = (참여율점수 × 0.4) + (바이럴점수 × 0.3) + (CPM점수 × 0.3)
```

## 🔍 FAQ

**Q: CPM 데이터는 정확한가요?**
A: 업계 평균 데이터를 기반으로 한 추정치입니다. 실제 수익은 채널, 시청자층, 시즌 등에 따라 달라질 수 있어요.

**Q: 왜 카테고리별로 CPM이 다른가요?**
A: 광고주들이 특정 카테고리(과학/기술, 교육 등)에 더 높은 광고비를 지불하기 때문이에요.

**Q: 수익 잠재력 점수는 어떻게 활용하나요?**
A: 같은 조회수라도 점수가 높은 영상 스타일이 더 수익성이 좋아요. 콘텐츠 기획 시 참고하세요!

**Q: 월 수익 시뮬레이션은 정확한가요?**
A: 현재 영상의 성과를 기준으로 한 예측이에요. 실제로는 채널 성장, 시즌성 등의 변수가 있어요.
